# Plan 010 — Docs idees: structure et index

## Objectif

- Ajouter une structure simple et durable pour `docs/06-idees/`.
- Mettre un index des idees existantes pour retrouver rapidement les sujets.
- Definir comment une idee passe de "note" a "chantier planifie" (`docs/90-plans/`).

## Decisions

- Un fichier `docs/06-idees/README.md` sert d'index + convention minimale.
- Les idees restent informelles (notes), mais doivent etre retrouvables.
- Quand une idee devient actionnable, on ouvre un plan `docs/90-plans/XXX-...`
  et on ajoute un lien depuis l'idee.
