# Builder de repas

## Contexte

- Composer un repas a partir d'une liste d'aliments peut etre long si l'ajout se fait uniquement via des formulaires.

## Proposition

- Dans l'editeur de repas:
  - afficher une bibliotheque d'aliments (recherche + filtres);
  - permettre de glisser/deposer un aliment dans le repas;
  - permettre d'ajuster rapidement la quantite (g / portion) apres ajout.

## Impact

- Saisie beaucoup plus rapide.
- UX plus agreable pour un usage quotidien.
