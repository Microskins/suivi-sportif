## Format d'une idee (minimal)

Copier/coller ce squelette dans un nouveau fichier si besoin:

```markdown
# Titre

## Contexte

- ...

## Proposition

- ...

## Impact

- ...

## Complexite

- S / M / L

## Liens

- Plan: docs/90-plans/XXX-... (quand c'est actee)
```

## Quand une idee devient un chantier

- Creer un plan dans `docs/90-plans/XXX-...`.
- L'indexer dans `docs/90-plans/README.md`.
- Ajouter un lien vers le plan depuis l'idee correspondante.
