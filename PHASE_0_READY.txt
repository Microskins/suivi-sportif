🎯 ÉTAPES D'EXÉCUTION - À FAIRE MAINTENANT
═══════════════════════════════════════════════════════════════════════════════

1️⃣  CRÉER LA STRUCTURE
    $ node setup-dirs.js

2️⃣  CONFIGURER .env
    $ cp .env.example .env
    (modifier si besoin)

3️⃣  DÉMARRER LES SERVICES
    $ docker-compose up -d

4️⃣  INSTALLER LES DÉPENDANCES
    $ npm install

5️⃣  VÉRIFIER TYPESCRIPT
    $ npm run typecheck

6️⃣  DÉMARRER LE SERVEUR
    $ npm run dev -w server
    (Terminal 1)

7️⃣  DÉMARRER LE CLIENT
    $ npm run dev -w client
    (Terminal 2)

8️⃣  VÉRIFIER LA SANTÉ
    $ curl http://localhost:3001/health
    $ open http://localhost:5173
