OPTION 1 - WINDOWS (One-Click Setup) ⭐⭐⭐

  Double-cliquez sur: setup.bat

  Ou via command line:
  .\setup.bat

OPTION 2 - TOUS LES SYSTÈMES
────────────────────────────────────────────
  1. Créer la structure:
     node setup-dirs.js

  2. Installer dépendances:
     npm install

  3. Configurer BD:
     npm run migrate -w server

  4. Démarrer:
     npm run dev

OPTION 3 - UNE SEULE LIGNE
────────────────────────────────────────────
  node setup-dirs.js && npm install && npm run dev

OPTION 4 - LINUX/MACOS
────────────────────────────────────────────
  bash setup.sh && npm install && npm run dev

🎓 STRUCTURE À CONNAÎTRE
═══════════════════════════════════════════════════════════════════════════════

Monorepo avec npm Workspaces:
  root/
    ├── server/         ← Workspace 1: Backend Fastify
    ├── client/         ← Workspace 2: Frontend React
    └── migrations/     ← Ressources partagées

Chaque workspace est indépendant:
  npm run dev -w server   (accès au workspace server)
  npm run dev -w client   (accès au workspace client)

Mais peuvent partager des dépendances au root si nécessaire.
