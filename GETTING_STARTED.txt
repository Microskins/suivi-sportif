🎯 PROCHAINES ÉTAPES - COMMANDES À EXÉCUTER
═══════════════════════════════════════════════════════════════════════════════

OPTION 1 : Setup AUTOMATIQUE (Recommandé pour Windows)
─────────────────────────────────────────────────────
  Double-cliquez sur : setup.bat
  ou dans PowerShell :  .\setup.bat

  ✓ Crée automatiquement la structure
  ✓ Installe toutes les dépendances
  ✓ Configure la base de données
  ✓ Lance npm run dev

OPTION 2 : Setup MANUEL
──────────────────────
  1. Créer la structure :
     node setup-dirs.js

  2. Installer les dépendances :
     npm install

  3. Configurer la base de données :
     npm run migrate -w server

  4. Démarrer le développement :
     npm run dev

OPTION 3 : Setup Bash (Linux/macOS)
────────────────────────────────────
  bash setup.sh
  npm install
  npm run migrate -w server
  npm run dev

═══════════════════════════════════════════════════════════════════════════════
✅ VÉRIFICATION - Une fois lancé avec "npm run dev"
═══════════════════════════════════════════════════════════════════════════════

1. Vérifier le serveur API :
   curl http://localhost:3001/health
   Réponse attendue: {"status":"ok","timestamp":"2024-04-24T..."}

2. Accéder au client React :
   http://localhost:5173

3. Tester les activités :
   curl http://localhost:3001/api/activities

═══════════════════════════════════════════════════════════════════════════════
📊 COMMANDES DISPONIBLES
═══════════════════════════════════════════════════════════════════════════════

npm run dev              Démarrer serveur + client simultanément ⭐
npm run dev -w server   Serveur uniquement
npm run dev -w client   Client uniquement
npm run build           Builder les deux projets
npm run start           Lancer en production
npm run test            Exécuter les tests
npm run lint            Vérifier la qualité du code
npm run migrate -w server  Exécuter les migrations BD
